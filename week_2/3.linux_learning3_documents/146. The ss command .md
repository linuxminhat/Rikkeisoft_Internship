146\. The ss command

Chủ đề: The ss Command (Socket Statistics)

ss (Socket Statistics) là một công cụ dòng lệnh trong Linux dùng để hiển thị thông tin chi tiết về các socket mạng. ss thay thế cho netstat trong các hệ thống hiện đại vì nó nhanh hơn và cung cấp thông tin chi tiết hơn. Socket là các điểm kết nối được sử dụng để giao tiếp giữa hai máy tính hoặc thiết bị qua mạng.

1. Cú pháp cơ bản

Để hiển thị tất cả các kết nối mạng đang hoạt động, chị chỉ cần chạy:

bash

Copy code

ss

Kết quả sẽ hiển thị danh sách các kết nối mạng, bao gồm cả địa chỉ nguồn, địa chỉ đích, và trạng thái của kết nối.

1. Các tùy chọn thường dùng với ss

Hiển thị tất cả các kết nối TCP:

Lệnh này hiển thị các kết nối TCP hiện đang hoạt động:

bash

Copy code

ss -t

Hiển thị tất cả các kết nối UDP:

Để kiểm tra các kết nối UDP đang hoạt động:

bash

Copy code

ss -u

Hiển thị cổng đang lắng nghe (listening ports):

Lệnh này sẽ hiển thị tất cả các socket đang "lắng nghe" trên hệ thống, bao gồm các cổng mở:

bash

Copy code

ss -l

Hiển thị kết nối TCP và UDP đang lắng nghe:

Chị có thể kết hợp nhiều tùy chọn để hiển thị các socket TCP và UDP đang lắng nghe:

bash

Copy code

ss -ltu

Hiển thị thông tin về mạng UNIX:

UNIX sockets là một dạng đặc biệt của socket, thường được sử dụng cho giao tiếp nội bộ giữa các tiến trình trên cùng một máy:

bash

Copy code

ss -x

Hiển thị số liệu thống kê về tất cả các kết nối:

Để hiển thị chi tiết các số liệu thống kê về các kết nối:

bash

Copy code

ss -s

Hiển thị các kết nối liên quan đến một cổng cụ thể:

Để xem các kết nối đang sử dụng một cổng cụ thể, ví dụ như cổng 80 (HTTP):

bash

Copy code

ss -t '( dport = :80 )'

Hiển thị các socket mở theo ứng dụng cụ thể:

Để liệt kê các kết nối đang sử dụng bởi một ứng dụng, chị có thể kết hợp với tên tiến trình hoặc user:

bash

Copy code

ss -p

1. Ví dụ thực tế

Hiển thị các kết nối TCP đang lắng nghe:

bash

Copy code

ss -lt

Hiển thị các kết nối UDP:

bash

Copy code

ss -u

Hiển thị các socket mở cùng với tên tiến trình đang sử dụng chúng:

bash

Copy code

ss -pt

Kết luận

ss là công cụ mạnh mẽ để quản lý và kiểm tra các kết nối mạng trên hệ thống Linux. Nó cung cấp thông tin chi tiết về các kết nối TCP, UDP, các cổng đang lắng nghe, và cả số liệu thống kê mạng, giúp dễ dàng kiểm soát và quản lý hệ thống mạng hơn so với công cụ netstat cũ.
