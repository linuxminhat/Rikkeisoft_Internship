Đây là lưu trữ dữ liệu trong Docker
Mặc định thì dữ liệu lưu trữ bên trong một container sẽ bị mất đi khi container đó bị xoá hoặc khởi động lại
CÁC PHƯƠNG PHÁP LƯU TRỮ DỮ LIỆU TRONG DOCKER :
1/Volumes (Ổ đĩa lưu trữ) :

- Được khuyến nghị
- Quản lí các volumes và lưu trữ chúng bên ngoài hệ thống file của container
  Cú pháp sử dụng :
  docker run -v my_volume:/path/in/container my_image
  2/Bind Mounts (Gắn kết thư mục) :
  Gắn một thư mục cụ thể từ hệ thống file của máy chủ host vào bên trong container
  Giúp container có thể truy cập trực tiếp vào dữ liệu máy chủ host
  Cú pháp sử dụng :
  docker run -v /path/on/host:/path/in/container my_image
  3/tmpfs Mounts: lưu dữ liệu tạm thời mà không cần lưu trữ nó ở trên đĩa
  Cú pháp sử dụng :
  docker run --tmpfs /path/in/container my_image
