Interactive Test Environments with Docker : Môi trường cách ly, giúp dễ dàng xoá bỏ khi đã hoàn tất các thử nghiệm
=> Hệ thống cài đặt cục bộ của bản thân có các phiên bản dependencies cũ và bản thân cần thử nghiệm các dependencies mới
=> Môi trường Interactive Test là môi trường cần thiết để có thể sử dụng nó !
Để bắt đầu môi trường thử nghiệm tương tác với image Python, bạn chỉ cần chạy lệnh sau:
-docker run -it --rm python
Giải thích :
-it : flag đảm bảo chạy container ở chế độ tương tác với terminal
--rm : tuỳ chọn này sẽ tự động xoá container sau khi nó dừng lại => tránh bị tồn đọng !
print
Để thoát gõ lệnh :
exit()
