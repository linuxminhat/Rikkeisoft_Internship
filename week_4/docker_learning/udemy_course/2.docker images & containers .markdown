Using & Runnng External (Pre-built) Images 
thùng chứa dựa trên image 
Cách các image được xây dựng sẵn từ cộng đồng hay các nguồn khác 
=> Pre buildt images là các image docker xây dựng sẵn và chia sẻ bởi dev khác
Nơi lưu trữ : Docker Hub
----
Cách dùng External Images :
- Tìm image trên Docker Hub
- Tải về : 
COMMAND : docker pull python:2
- Chạy
COMMAND : docker run -p 3000:3000 python2
COMMAND : docker run -it node : 
-it : tổ hợp tham số 
-i : nhập lệnh terminal 
-t : tương tác với container
